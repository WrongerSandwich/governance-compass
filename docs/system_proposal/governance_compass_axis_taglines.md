# Axis Taglines — Quick Reference

Short taglines for each axis, designed to work as subtitle text in the homepage axis list and as inline descriptions in the results page axis breakdown.

## Usage

- **Homepage:** Display below each axis name in the axis list
- **Results page:** Display as a text-secondary line below the axis name in each breakdown row
- **Styling:** 11–12px, text-tertiary or text-secondary, below the axis name
